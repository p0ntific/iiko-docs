# Окно заказа (ресторанный режим)

## Краткое описание

Основной интерфейс для создания и управления заказами в ресторанном режиме. Предоставляет полный функционал для приема заказов, работы с модификаторами, управления курсами подачи и координации с кухней.

## Функциональные элементы

### Основные компоненты страницы

- **Информация о столе** - номер, количество гостей, время посадки
- **Меню товаров** - категории и позиции для заказа
- **Состав заказа** - список выбранных блюд с деталями
- **Панель управления** - основные действия с заказом
- **Информация о курсах** - группировка блюд по подачам
- **Модификаторы** - дополнения и изменения к блюдам

### Функциональность

- Создание и редактирование заказов
- Работа с модификаторами блюд
- Управление курсами подачи
- Расчет стоимости заказа
- Отправка заказов на кухню
- Печать пречеков и счетов

## Бизнес-правила

### Создание заказа

1. Выбор блюд из меню
2. Настройка модификаторов
3. Управление курсами
4. Расчет стоимости

### Управление количеством и порциями

- Увеличение/уменьшение количества порций
- Дробные порции (0.5, 1.5 порции)
- Деление блюда между гостями
- Детские порции с соответствующими ценами

### Курсы подачи

- Аперитивы - напитки до еды
- Холодные закуски - салаты, нарезки
- Супы - первые блюда
- Основные блюда - горячие позиции
- Десерты - сладкие блюда
- Напитки - кофе, чай, алкоголь
